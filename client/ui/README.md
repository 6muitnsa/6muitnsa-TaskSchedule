# TaskSchedule UI

## 项目结构
```
ui/
├── src/                    # 源代码目录
│   ├── api/               # API接口封装
│   ├── components/        # 可复用组件
│   ├── styles/           # 全局样式文件
│   ├── views/            # 页面视图组件
│   ├── App.vue           # 根组件
│   └── main.js           # 入口文件
├── index.html            # HTML模板
├── vite.config.js        # Vite配置文件
├── package.json          # 项目依赖配置
└── quick_start.bat       # 快速启动脚本
```

## 环境配置
- 环境变量配置已移至 `../config/.env`
- 其他配置文件位于 `../config/` 目录
